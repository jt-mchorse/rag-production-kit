# Contributing

This repo is part of JT McHorse's AI/ML engineering portfolio. It follows the protocol in the [portfolio handoff](https://github.com/jt-mchorse/portfolio-ops/blob/main/COWORK_HANDOFF.md).

- Sessions are issue-driven; no work without a tracking issue.
- ~60-minute session cap with mandatory memory updates (see `.skills/portfolio-session/`).
- All changes via PR; main is protected.
